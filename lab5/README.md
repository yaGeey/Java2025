# Лабораторна робота №5 — Наслідування та поліморфізм

**Наслідування (Inheritance)** - це механізм об'єктно-орієнтованого програмування, що дозволяє створювати новий клас на основі існуючого. Клас-нащадок успадковує всі поля та методи батьківського класу і може додавати свої власні або перевизначати успадковані. В Java використовується ключове слово extends.

**Поліморфізм (Polymorphism)** - здатність об'єктів різних класів обробляти дані різними способами через єдиний інтерфейс. В Java реалізується через перевизначення методів (override) та можливість використання змінної батьківського типу для посилання на об'єкти дочірніх класів.

**Абстрактний клас (Abstract class)** - клас, який не може мати екземплярів і служить базою для інших класів. Може містити абстрактні методи (без реалізації), які обов'язково повинні бути реалізовані в класах-нащадках. Оголошується з ключовим словом abstract.

**Перевизначення методів (Method Overriding)** - можливість в класі-нащадку змінити реалізацію методу, успадкованого від батьківського класу. Використовується анотація @Override. Сигнатура методу має бути ідентичною.

**Інтерфейс Comparable** - стандартний Java інтерфейс, що дозволяє об'єктам класу порівнюватись між собою. Містить метод compareTo(), який визначає природний порядок сортування об'єктів.
