## Лабораторна робота №1 — Масиви

### Завдання
Реалізувати клас `Matrix`, який дозволяє працювати з двовимірною матрицею чисел типу `long`.  
Передбачити методи для транспонування матриці (`transpose`) та знаходження суми мінімальних елементів кожного стовпця (`getSumOfMinimalElementsPerColumn`)

### Використані можливості Java
- Класи та об'єкти
- Модифікатори доступу (`private`, `public`)
- Конструктор класу
- Перевірка вхідних даних (`if`, `throw IllegalArgumentException`)
- Цикли `for`
- Метод `System.arraycopy` для копіювання масивів
- Методи класу `Math` (`Math.min`)
